# 🚀 MernLearning

> Plataforma de aprendizaje MERN Stack con revisión automática de código por IA (Google Gemini)

## 📚 Descripción

MernLearning es una plataforma educativa completa para aprender el stack MERN (MongoDB, Express, React, Node.js) de forma progresiva e interactiva. Incluye un editor de código integrado con revisión automática por inteligencia artificial usando Google Gemini (gratuito).

## ✨ Características

- 🤖 **Revisión IA** con Google Gemini (gratis)
- ⚡ **Editor de código** Monaco integrado
- 🏆 **Sistema XP/Niveles** gamificado
- 🔐 **Autenticación** JWT + Google OAuth
- 📱 **Diseño responsivo** moderno
- 🗺️ **5 módulos** progresivos
- 🔓 **Desbloqueo automático** al 70% completado

## 🗺️ Módulos

| # | Módulo | Descripción |
|---|--------|-------------|
| 1 | 🌐 HTML & CSS | Estructura y estilo web |
| 2 | ⚡ JavaScript | Programación web interactiva |
| 3 | ⚛️ React | Frontend moderno |
| 4 | 🟢 Node.js & Express | Backend con API REST |
| 5 | 🍃 MongoDB | Base de datos NoSQL |

## 🚀 Inicio Rápido

Ver [SETUP.md](docs/SETUP.md) para instrucciones detalladas.

```bash
# Backend
cd backend && npm install && npm run dev

# Frontend (otra terminal)
cd frontend && npm install && npm start
```

## 🛠️ Stack Tecnológico

**Backend:** Node.js, Express, MongoDB, Mongoose, JWT, Passport.js, Google Gemini  
**Frontend:** React 18, TypeScript, Monaco Editor, React Router v6, Axios
